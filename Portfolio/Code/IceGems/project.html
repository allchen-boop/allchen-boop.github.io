Mini Maya : C++

I made a mini version of Autodesk's Maya! This version includes shader based skin deformation, an interactive skeleton, obj file importing, a skeleton JSON file reader, interactive camera, catmull-clark subdivision, and topology editing functions.

The gui allows users to view all vertices, half edges, and faces of the object. Additionally, users can toggle the vertex positions, face colour, and joint position and rotations. There are also buttons to add vertices, triangulate, subdivide, and skin the mesh.
